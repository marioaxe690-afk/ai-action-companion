# AI Action Companion

AI Action Companion 是一个“AI 主动陪伴式行动助手”的黑客松赛前网页原型。它面向容易拖延、目标启动困难的大学生，帮助用户把模糊目标拆成极小、可执行、可以立刻开始的行动步骤。

## 项目定位

这是一个可交互的前端原型，用来验证产品概念、页面信息架构和核心交互路径。当前版本不连接真实 AI 服务，也不包含后端能力。

## 目标用户

- 容易拖延但希望开始行动的大学生
- 有学习、申请、运动、生活管理等目标，但不知道第一步怎么做的人
- 需要陪伴感、提醒感和轻量反馈，而不是复杂项目管理工具的人

## 核心痛点

- 目标太大，用户难以启动
- 计划太抽象，第一步不够具体
- 独自执行时缺少反馈和陪伴
- 记录负担太重，导致坚持成本变高

## 当前原型功能

- 输入一个模糊目标
- 点击示例目标快速填入输入框
- 生成 mock 行动链拆解结果
- 切换严格导师、温柔朋友、猫娘陪伴、冷静监督者四种人设
- 勾选行动步骤并同步更新完成数量、进度条和今日完成率
- 根据当前人设和完成进度展示不同陪伴反馈
- 展示产品流程框架：目标输入 → AI 理解 → 极小任务拆解 → 人设陪伴 → 反馈记录 → 下一步调整

## 技术栈

- Vite
- React
- TypeScript
- CSS

## 如何运行

```bash
npm install
npm run dev
```

打开终端输出的本地地址即可查看页面。生产构建命令：

```bash
npm run build
```

## 当前不做什么

- 不接入真实 AI 服务
- 不做后端
- 不做登录注册
- 不做数据库
- 不做移动端应用
- 不做后台监控
- 不做自动截屏
- 不包含任何真实服务凭据

## 后续计划

- 根据真实学生访谈调整目标示例和行动链文案
- 增加更细的人设语气和反馈状态
- 验证“极小行动”是否能提升启动率
- 设计轻量记忆机制，用于记录用户偏好、常见拖延点和历史完成情况
- 在黑客松后再评估是否接入真实 AI 拆解能力
