# 刷到即行动：AI 内容行动化助手

刷到即行动是一款面向大学生和年轻创作者的 AI 内容行动化助手。它把用户刷到、看到、想到的内容，转化为马上能执行的下一步行动。

当前项目是 Vite + React + TypeScript 前端原型，使用 mock 数据演示核心流程，不接真实 AI API，不包含后端、数据库、登录注册或任何密钥。

当前原型采用暗色 AI 玻璃工作台风格，用于强化黑客松展示感和 AI 产品气质。

## 产品定位

普通 AI 给用户一段答案，刷到即行动把内容变成行动卡片。普通 Todo 要用户自己想清楚任务，刷到即行动从内容里自动提炼任务。

用户可以输入短视频文案、截图通知、学习资料、比赛信息、项目灵感或生活目标。系统会先理解内容场景，再重构关键信息，最后生成可执行的行动卡片、任务链和下一步计划，并通过个性化陪伴与轻量记忆持续推动用户完成。

## 当前原型结构

- 顶部差异化 Hero：对比普通 AI 和刷到即行动
- 产品定位区：说明 AI 内容行动化助手的转化边界
- 内容输入中心：支持比赛通知、学习内容、校园通知、短视频文案、项目灵感、生活目标
- AI 场景理解区：展示 mock 的内容类型、用户意图、推荐转化、风险判断和适合赛道
- 行动卡生成区：生成 5 张可勾选的行动卡，并同步更新完成进度
- 陪伴执行区：保留严格导师、温柔朋友、冷静监督者、灵感搭子四种人设
- 记忆沉淀区：展示 mock 用户行动画像，强调“下次更懂你”

## 第一版深做场景

第一版重点高亮“比赛通知”，以 AI 黑客松通知为主示例，演示从比赛信息到行动卡、MVP 边界、团队分工和路演故事的完整转化链路。

## 如何运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run typecheck
npm run build
```

## 当前不做事项

- 不接真实 AI API
- 不添加后端
- 不添加数据库
- 不添加登录注册
- 不写任何 API Key、Token、secret
- 不做图片真实识别
- 不做视频抓取
- 不做抖音数据抓取
- 不做浏览器插件
- 不引入复杂依赖
