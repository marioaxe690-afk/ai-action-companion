# Roadmap

## Now

- 完成 Vite + React + TypeScript 前端原型
- 搭建核心页面结构和 mock 数据
- 实现目标输入、示例填入、人设切换、行动链勾选和进度反馈
- 明确当前版本边界：只做前端交互，不接真实服务

## Next

- 优化行动链拆解逻辑，让不同目标显示更贴近语义的 mock 步骤
- 增加更多陪伴反馈状态，例如卡住、连续完成、中途放弃
- 做一轮大学生可用性访谈，验证文案、流程和视觉负担
- 补充基础组件测试或交互测试
- 准备黑客松演示脚本和展示素材

## Later

- 评估真实 AI 拆解能力的接入方式
- 设计轻量记忆和偏好记录
- 探索提醒机制、复盘机制和连续行动机制
- 增加多目标管理和一周行动回顾
- 根据验证结果决定是否继续做后端、账号和持久化
